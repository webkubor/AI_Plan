## **武功体系** 

### **衔接建议（体系完整性）**

✅ **修为等级 = 你能“打出多强”**

✅ **轻功四境 = 你能“移动多快”**

✅ **武道领悟 = 你是否“越阶战敌”**

✅ **武学门类 = 你用“什么武器、什么风格”来战斗**

#### **【修为等级】—— 表示整体实力层级，决定你“能到什么高度”**

| **等级层级** | **子阶段**  | **实力特征**                                     |
| ------------ | ----------- | ------------------------------------------------ |
| **不入流**   | -           | 徒有蛮力，连“气”都未成，江湖最底层               |
| **三品**     | 初阶 → 极境 | 内力初成，初识气感，力可断金碎石                 |
| **二品**     | 初阶 → 极境 | 内劲外放，气可成芒，攻势已具杀伤                 |
| **一品**     | 初阶 → 极境 | 暗劲可控，罡气内藏，动可致命                     |
| **宗师**     | 初阶 → 极境 | 入“势”之道，控场压敌，技法简而精                 |
| **大宗师**   | 初阶 → 极境 | 悟“意”之真，武意通神，一念生杀                   |
| **天人**     | 初阶 → 极境 | 破碎成规，天人合一，掌天地之势、意动生威（极少） |

>  **备注：每一大阶可细分为：初阶 / 小成 / 大成 / 大圆满 / 极境**

#### **💨【轻功四境】—— 表示你“身法速度”的精妙程度**



燕行，踏雪，凌空，无影

| **轻功境界** | **描述精要**                         |
| ------------ | ------------------------------------ |
| **燕行逸踪** | 仿若灵燕剪波，轻捷无声，奔行如风     |
| **踏雪无痕** | 雪地留行无迹，身轻如烟，敌难捕影     |
| **凌空虚渡** | 御气腾空，空中滞留，腾挪如鹤舞苍穹   |
| **无影幻形** | 身形虚实莫测，动若雷电星芒，忽现忽灭 |

> 注：轻功并不等于修为，常有低阶高手以“轻功领悟”越阶胜敌，如“花影迷踪步”、“碧海云踪”等为天下三大轻功。

### **【五力之道】—— 表示你“力量理解深度”的核心维度**

> 这是贯穿整个武功修为的“底层逻辑体系”，对应你真正的战斗本质认知。

| **层级** | **名称** | **含义**               | **表现形式**               | **常见领悟者**       |
| -------- | -------- | ---------------------- | -------------------------- | -------------------- |
| ①        | **气**   | 内力之本源，动能流转   | 运功、调息、气力贯通       | 三品、二品打基础     |
| ②        | **芒**   | 气的锋锐延伸，具杀伤力 | 剑气、掌风、枪芒等外放气劲 | 二品中后期           |
| ③        | **罡**   | 暗劲凝聚，内力深层     | 贴身穿透、劲震脏腑         | 一品高手常用         |
| ④        | **势**   | 控局压人，气势为主     | 压制节奏、一敌多也不乱     | 宗师起步者           |
| ⑤        | **意**   | 心中之道，精神信念     | 无招胜有招，武意外显       | 大宗师，或超凡者起步 |

**对应口诀简明总结：**

- **气为源，芒为刃，罡为杀，势为控，意为神。**
- **初学者炼气，中乘者用罡，高手控势，巅峰者悟意。**

### **🧠【武道领悟】—— 决定你是否能“越阶战敌”的关键**

| **类型**     | **表现形式**                 | **可能的突破效果**                     |
| ------------ | ---------------------------- | -------------------------------------- |
| **气贯周身** | 内力游走无滞，调动如意       | 内功、护体、恢复速度大幅增强           |
| **芒可伤敌** | 攻击气劲外放，破甲透体       | 杀伤增强，可远距伤敌                   |
| **罡气藏锋** | 暗劲凝练，一击致命           | 武器或身体可蓄杀机，贴身搏杀占尽优势   |
| **势控全局** | 战场节奏主导，对手动作被预判 | 群战、困敌、反打极强，甚至“未打先胜”   |
| **意动则威** | 精神化形，技随心转           | 无招胜有招，越阶战敌、秒杀对手成为可能 |

>  特别说明：你不必等境界到了“宗师”才可悟“势”，也不必等“大宗师”才有“意”，只要领悟足够深，你哪怕三品也能越阶战宗师！
