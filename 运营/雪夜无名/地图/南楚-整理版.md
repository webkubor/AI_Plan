# 南楚王国

## 地理概况

南楚位于大陆中部偏南，是三国中最为繁华富庶的国家，拥有丰富的水系和肥沃的土地。

## 重要城池与地点

### 丹阳城（皇都）

位置：南楚中央，天子脚下，权力与阴谋交织之地，江湖人眼中的"龙渊腹地"。

#### 重要地点
- **皇城**：楚帝寒江淮的居所，权力中心
- **流月楼**：全南楚最好的酒楼，建在丹阳城北，倚水而立，楼高五层
- **江心酒楼**：公主寒子瑜所有，立于丹阳江畔最深水道中央，四面环江
- **湖光小谢**：寒子瑜直属，丹阳城西，碧波环绕，一座湖心小榭
- **松云居客栈**：装修复古，低调，清净之地（西燕的探子窝点）
- **三山镖局**：丹阳最大镖局，由退伍铁血卫创立，镇局之宝"封山印"
- **梧桐书楼**：王公子弟、才子骚人聚集地，设赌诗、押赋之风
- **百子巷赌庄**：丹阳城西南角，偏离主街，地势低洼
- **鸳鸯坊**：南楚最神秘、最尊贵的青楼，四大名坊之首
- **听潮酒肆**：江湖人喜欢去的地方
- **湖光小榭**：丹阳城西，占地约百方丈，分为水榭四司、风月六坊、执令三部

### 湘州城

位置：南楚皇城东北方，与北齐接近

#### 重要地点
- **青莲寺**：湘州城以北六十里
- **大石村**：主角从小长大的地方（也有个松云居客栈-伏笔）
- **云来客栈**：主城最大客栈，江湖客聚集，常有赌斗、挑战
- **松云居客栈**：装修复古，低调，清净之地（西燕的探子窝点）
- **烟雨赌坊**：用兵器、秘术、神通下注，暗设禁术擂台
- **破月馆**：湘州外城，一座废弃旧庙后殿所改，隐匿深巷

### 云梦城

位置：南楚皇城南方，四季雾霭缭绕，终年不散，昼夜难分，**梦泽之都**

#### 重要地点
- **瑶光宫**：云梦城和丹阳城之间（顶级势力）
- **寒绝谷**：在百花谷外十余里，有一处寒潭，四面青石垂壁
- **镜水台**：据说能"照人命格"，乃南楚命理最神秘之地
- **月华阁**：修道女子清修场所，琴技御心术，两层境界评定

### 江陵城

位置：南楚皇城西方，"商贾重镇"，贸易繁盛，暗流涌动，藏龙卧虎

#### 重要地点
- **十三香楼**：极尽奢靡，金钱至上，老鸨金三娘
- **赤风镖局**：西域背景，掌多条西向线路，与北齐、西燕皆通
- **听雨阁**：消息情报买卖的地方，暗地里贩卖人口，负责人宁三爷

### 白帝城

位置：南楚西北之地，西侧是西燕，北侧是北齐，战略要地

#### 重要地点
- **白帝城城门**：北门通北齐、西门通西燕，城墙厚重千丈
- **魁星台**：军民共用擂台，有"百战未死者"封号
- **清平镇·来福客栈**：热闹市井客栈，酒香扑鼻
- **松云居客栈**：西燕探子窝点，低调、复古，房间布局复杂，暗道纵横
- **红袖楼**：城内知名妓院，也兼做江湖情报交换地
- **隐龙巷**：小巷幽深，暗藏江湖帮派据点
- **百胜馆**：南楚武馆，传授轻功与刀法
- **赤月岭**：旧时魔宗的**赤月古殿**

### 洛水城

位置：南楚皇城东南方，"水乡柔肠"，诗意与杀意共存之地

#### 重要地点
- **梦蝶楼**：小楼依水而筑，清雅朴素，不招牌匾
- **小舟居**：画舫，每日文人雅士吟诗斗赋，月夜琴音，水面杀机
- **青羊镇**
- **天剑峰**

### 清洛城

位置：南楚皇城东方，富贵之地，家族势力林立，权贵横生

#### 重要地点
- **公孙府**：权倾朝野，兵权、财权并掌，金碧辉煌
- **玉堂书会**：官家子弟聚会之地，隐有党争气氛
- **红香窟**：只接"东南五贵"之客，女子皆习音杀术

### 南楚四大关

1. **白钰关**（北境第一关）：位于南楚最北，关城以白石砌成，远望若玉
2. **天岚关**（西北屏障）：关城建在群山之间，常年云雾缭绕，似天岚飘渺
3. **青枫关**（西境要塞）：位于通往西燕的山谷间，谷内多枫林
4. **丹川关**（南楚最后的屏障）：位于南北交通咽喉，南楚的粮道军械多汇于此

## 人物列表

### 皇室成员

- **寒江淮**：楚帝，南楚统治者
- **寒子易**：大皇子，兵马
- **寒子旭**：二皇子，权谋
- **寒子瑜**：公主，拥有江心酒楼，湖光小谢直属于她

### 朝廷官员与势力

- **吴江**：大监
- **华清风**：御医
- **萧靖**：铁血卫统领
- **影**：暗卫首领

- 【锐锋营】黑色虎纹皮甲，虎头银刀，背负弓弩（皇帝嫡系，精锐单兵，冲锋与远程兼备）
- 【神机营】红色豹纹皮甲，腰挂锁链银钩，鹰头银刀（白帝城，大将军-赵凌岳，阵中支援与奇袭，灵活多变）
- 【飞羽骑】轻便白色骑甲，羽毛装饰，大漠弯刀、长枪（白帝城，大将军-赵凌岳，机动精锐）
- 【铁血卫】深红铠甲，血虎纹章，玄铁大刀、黑虎盾（首领：萧靖，禁军重防守力量）

**锐锋令本身**：

- 百锐锋令 → 黑铁打造，背面刻“锐锋”二字。
- 千锐锋令 → 赤铜铸造，令身厚重，边缘有虎纹。
- 万锐锋令 → 黄金镶玉，正面是“锐锋令”，背面篆刻“皇”字。

### 江湖人物

- **金三娘**：十三香楼老鸨
- **宁三爷**：听雨阁负责人，暗地里贩卖人口

### 其他重要人物

- **李牧**（化名慕青寒）：主角，李青衣之子
- **李青衣**：天下三绝之一，主角之父
