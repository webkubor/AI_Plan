按歌曲从开头到结尾的常规顺序重新排列，并梳理了清晰的中文译名与核心作用：

|  英文名称  |  中文常用译名   |                           核心作用                           |
| :--------: | :-------------: | :----------------------------------------------------------: |
|   Intro    |   引言 / 前奏   |         歌曲的开头部分，用于引入主题、旋律或情感氛围         |
|   Verse    |   主歌 / 诗歌   |          歌曲的主体部分，通常包含故事情节和核心主题          |
| Pre-Chorus | 前副歌 / 预副歌 |          衔接主歌与副歌，为副歌做情绪和旋律上的铺垫          |
|   Chorus   |   副歌 / 合唱   |      歌曲的高潮部分，包含主旋律和主题的重复，记忆点最强      |
|   Bridge   |  桥段 / 过渡段  |        连接不同段落，通常引入新的主题或情感，避免单调        |
| Interlude  |   间奏 / 插段   | 两个主要部分之间的短暂插入，一般无歌词，用于过渡或引入新元素 |
|    Hook    |  钩子 / 记忆点  |    歌曲中最具吸引力的部分，通常是副歌或重复频率较高的段落    |
|   Outro    |   尾奏 / 尾声   |             歌曲的结束部分，可逐渐减弱或突然收尾             |
