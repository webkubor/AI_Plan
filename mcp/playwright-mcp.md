## playwright-mcp

https://github.com/microsoft/playwright-mcp

> chrome-devtools-mcp, 只能做些比较简单的操作
>
> playwright MCP 的核心价值是 “**让 AI 真正‘懂’浏览器，让非技术人员也能掌控自动化**”

```js
{
  "mcpServers": {
    "playwright": {
      "command": "npx",
      "args": [
        "@playwright/mcp@latest"
      ]
    }
  }
}
```
