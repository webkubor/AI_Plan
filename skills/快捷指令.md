# 快捷指令：让所有 AI 共用"大脑"

## 为什么需要

不同 AI 工具对"长期记忆"的支持不同：

| AI 工具 | 是否自带长期记忆 | 问题 |
|----------|----------------|------|
| Gemini | ✅ 支持（save_memory） | 直接可用 |
| Codex | ❌ 不支持 | 每次都要手动加载规则 |
| Claude Code | ❌ 不支持 | 每次都要手动加载规则 |

如果不处理，每次用 Codex/Claude 都要先手动敲 `/start` 或说明规则，效率很低。

## 解决方案：Shell 别名

通过在 `~/.zshrc` 中配置 alias，让启动命令自动带上加载规则的参数。

### Codex

```bash
# 创建别名：启动时自动执行 /start 技能
alias codex-vibe='codex "/start"'
```

### Claude Code

```bash
# 创建别名：启动时自动加载 AI_Common
alias claude-vibe='claude --context ~/Documents/AI_Common'
```

### 使用方法

配置完成后，新开终端：

```bash
# 正常启动 Codex（不加载规则）
codex

# 启动 Codex 并自动加载规则
codex-vibe

# 启动 Claude 并自动加载规则
claude-vibe
```

## 配置步骤

1. 编辑 `~/.zshrc` 文件
2. 添加上面的 alias 代码
3. 保存并执行 `source ~/.zshrc` 让配置生效
4. 新开终端即可使用

## 原理

- **`/start`** 是 Codex 的一个 Skill，定义在 `~/.codex/skills/start/SKILL.md` 中
- 当执行 `codex "/start"` 时，Codex 会自动读取该 Skill 文件
- 该 Skill 文件会读取 `AI_Common/index.md` 并加载所有规则

这样就实现了：一键启动 = 自动加载"大脑"
