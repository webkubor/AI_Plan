 1. Frontend Design Skill (刚才安装的那个)

    https://skillsmp.com/zh/skills/anthropics-claude-code-plugins-frontend-design-skills-frontend-design-skill-md
   * 核心理念：“风格矫正”。它是一套原则，强迫 AI 打破平庸，追求特定的艺术风格（如野兽派、极简主义）。
   * 工作方式：内化。通过 Prompt 改变 AI 的思考方式，让 AI 像个“挑剔的设计师”一样思考。
   * 优势：极具个性，适合想要“独特”、“不撞衫”的设计。
   * 数据量：较小，主要靠 AI 自身的创造力发挥。

  2. UI UX Pro Max (新发现的这个)

     https://github.com/nextlevelbuilder/ui-ux-pro-max-skill
   * 核心理念：“设计智能数据库”。它是一个庞大的知识库，提供经过验证的、成套的设计方案。
   * 工作方式：检索 + 组装。
       1. 分析需求：分析你的 Prompt（如“宠物美容落地页”）。
       2. 检索数据库：从 57种 UI 风格、95种色板、56种字体组合、14种落地页模式中匹配最佳组合。
       3. 生成代码：基于匹配到的组合生成代码。
   * 支持技术栈：明确支持 React, Next.js, Vue, Svelte, SwiftUI, React Native, Flutter, Tailwind 等 8 种技术栈。
   * 优势：专业、全面、不出错。适合快速构建高质量、标准化的商业产品。它能确保你用对配色、字体和组件库。
